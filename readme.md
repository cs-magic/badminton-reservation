# Uni Pusher - 统一社交平台发布工具

Uni Pusher 是一个便捷的多平台内容发布工具，支持一键将想法同步发布到多个社交平台。

## 功能特性

### 核心功能
- 支持多平台同步发布
  - 微信朋友圈
  - 即刻
  - 知识星球
  - (未来可扩展更多平台)
- 富媒体内容支持
  - 文本内容
  - 图片上传（支持多图）
  - 链接预览
- 平台特定功能适配
  - 分组可见性设置
  - 地理位置标记
  - 话题/圈子选择
  
### 用户体验
- 统一的发布界面，智能适配各平台特性
- 实时预览各平台发布效果
- 草稿保存与历史记录
- 发布状态追踪

## 技术架构

### 前端技术栈
- Next.js + React
- TailwindCSS
- React Query
- Zustand (状态管理)

### 后端技术栈
- Node.js
- PostgreSQL (数据存储)
- Redis (缓存)
- AWS S3 (图片存储)

### 核心模块
1. 平台认证模块
   - OAuth 授权
   - Token 管理
   
2. 内容编辑器
   - 富文本编辑
   - 图片上传与处理
   - 平台预览

3. 发布管理
   - 队列系统
   - 失败重试
   - 状态追踪

4. 数据同步
   - 平台 API 适配
   - 内容转换
   - 媒体处理
